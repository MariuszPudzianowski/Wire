An application implementing the WireWorld cellular automata. Basic rules for the automata along with
application functionality are described in the file 'Specyfikacja' in Polish.

This application's code is functional, but of poor quality. If we were to code it today, we obviously
wouldn't have done it this way. However, it's not worth refactoring, as a majority of the code would have to
be rewritten. Having said that, adding new Wire object types or simple functionality is still pretty easy.
